This program generates quality control reports for RAW mass spectrometry data files.

This project uses:
* **ThermoRawFileParser** for RAW to mzML conversion. [Hulstaert et al., JPR (2020)](https://pubs.acs.org/doi/10.1021/acs.jproteome.9b00328)
* **MSFragger** for the search. [Kong et al., Nature Methods (2017)](https://www.nature.com/articles/nmeth.4256)
